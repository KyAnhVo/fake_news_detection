Goal

My website provides a free way to categorize different news articles in different categorization. It is either fake news detection, politically polarized left/right detection, etc.

This project was created as a for-fun project and to learn about how to use different ML models to analyze texts, including Naive Bayes, Linear Regression, Support Vector Machine, and Neural Network.
